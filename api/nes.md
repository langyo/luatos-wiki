## nes.init(file_path)



nes模拟器初始化

**参数**

|传入值类型|解释|
|-|-|
|string|file_path 文件路径|

**返回值**

|返回值类型|解释|
|-|-|
|bool|成功返回true,否则返回false|

**例子**

```lua
nes.init("/luadb/super_mario.nes")

```
