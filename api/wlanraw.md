## wlanraw.setup(opts, cb)



初始化WLAN的RAW层

**参数**

|传入值类型|解释|
|-|-|
|table|opts 配置参数|
|function|回调函数,形式function(buff, size)|

**返回值**

|返回值类型|解释|
|-|-|
|boolean|true表示成功,其他失败|

**例子**

```lua
-- 当前仅XT804系列支持, 例如 Air101/Air103/Air601/Air690
wlanraw.setup({
    buffsize = 1600, -- 缓冲区大小, 默认1600字节
    buffcount = 10, -- 缓冲区数量, 默认8
}, cb)

```
